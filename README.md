#### Run project: npm run dev
#### http://localhost:3000/ 
(concurrently run with shoeshop-backend : http://localhost:8080/)

#### Các tính năng:
* Customer:
  + Sign up / Sign in / Sign out
  + View detail product's information, profile, cart
  + Edit, delete product in cart
  + Update avatar
  + Add product to cart
    
* Admin:
  + Sign in / Sign out
  + CRUD users, products
